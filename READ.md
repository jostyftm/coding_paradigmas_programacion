## 🚀 Cómo Empezar (Instalación y Despliegue)

Sigue estos pasos para descargar el proyecto y levantar el entorno de desarrollo seguro (Mozart 1.4.0) en tu máquina local. 

### 1. Clonar el repositorio
Abre tu terminal y ejecuta el siguiente comando para descargar el código fuente:
```bash
git clone git@github.com:jostyftm/coding_paradigmas_programacion.git
```

### 2. Ingresar al directorio del proyecto
```bash
cd tu-repositorio
```

### 3. Levantar el entorno con Docker Compose
```bash
docker-compose up -d
```
